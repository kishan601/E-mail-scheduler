## Packages
date-fns | Date formatting for tables and schedules
papaparse | Client-side CSV parsing for email lists
framer-motion | Smooth animations for UI transitions
recharts | Visualization for email statistics
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
Authentication is handled via Replit Auth (/api/login, /api/logout)
CSV parsing happens client-side; only valid emails are sent to backend
Backend expects ISO strings for dates
